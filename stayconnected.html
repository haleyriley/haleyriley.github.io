Staying Connected Website

This was a class project. I designed, illustrated, and coded the website using HTML, CSS, and Javascript.

I chose to make a website that features various free ways to stay connected with friends and loved ones during quarantine. I thought it would be nice to have a bunch of different resources all in one place, so that you donâ€™t have to click around to search through different websites.

I chose a space-y theme, because it feels like we are all kind of floating through space alone right now trying to reconnect with the regular world. I made the font using my friends handwriting. It was fun creating a typeface and finding a way to use it out in the world.
